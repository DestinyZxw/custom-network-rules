# custom-network-rules 

#### 简单描述  
使用 https://acl4ssr-sub.github.io/ 规则转换  
1,点击“进阶模式”  
2,导入订阅链接（机场、自建的）  
3,远程配置把这个仓库里面的“.ini”结尾的文件地址填上去  
4,转换导入clash即可  

### ini里添加自己的rules  
#### 1,rules 提供的对应标准集合  
1,https://github.com/blackmatrix7/ios_rule_script   
#### 2,ini中使用的rules地址，暂不确定具体是在哪个git库中维护
